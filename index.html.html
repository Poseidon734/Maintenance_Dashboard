Linear Regression
Predice una variable continua (Ej: vida útil restante RUL) como función lineal de variables de entrada. Simple pero efectivo como baseline.
y = β₀ + β₁x₁ + β₂x₂ + ε
✦ Interpretable y rápido
✦ Sensible a outliers
✦ Asume linealidad
Random Forest
Ensemble de árboles de decisión para clasificación (falla/no falla) o regresión. Robusto al overfitting y con feature importance nativo.
ŷ = (1/N) Σ hₙ(x)
✦ Maneja no linealidades
✦ Feature importance
✦ Alto desempeño
LSTM / GRU
Redes recurrentes para secuencias temporales. Capturan dependencias a largo plazo en series de vibración, temperatura o corriente.
hₜ = f(Wₕhₜ₋₁ + Wₓxₜ + b)
✦ Memoria temporal
✦ Ideal para sensores
✦ Requiere más datos
SVM / SVR
Support Vector Machines para clasificación y regresión. Efectivo en alta dimensionalidad con datos limitados usando kernels RBF o polinomiales.
f(x) = sign(Σ αᵢyᵢK(xᵢ,x) + b)
✦ Eficiente con pocos datos
✦ Kernels flexibles
✦ Lento en datos grandes
Autoencoder
Red neuronal que aprende representación comprimida. El error de reconstrucción es score de anomalía sin datos etiquetados.
x̂ = dec(enc(x)) ; score = ‖x - x̂‖
✦ No supervisado
✦ Detección de anomalías
✦ Interpretación compleja
XGBoost / LightGBM
Gradient boosting optimizado. Estado del arte en tabular data para predicción de fallos, RUL y clasificación de modos con SHAP explicability.
F(x) = Σ γₖ · 𝟙[xⱼ < sₖ]
✦ Mejor rendimiento tabular
✦ SHAP explicability
✦ Rápido y escalable

Guía de Extracción — SAP PM PASO A PASO
1
Acceder a IW38 — Lista de Órdenes de Mantenimiento
En el menú SAP: Mantenimiento de Planta → Procesamiento de Órdenes → Lista de Órdenes (IW38). Filtra por centro de planificación, tipo de orden (PM01=Correctivo, PM03=Preventivo) y rango de fechas.
2
Seleccionar variante de visualización con campos clave
Agregar columnas: Equipo (EQUNR), Denominación (EQKTX), Puesto Trabajo (ARBPL), F.inicio real (GSTRP), F.fin real (GLTRP), Horas reales (ISMNG), Clase OT (AUART), Causa avería (URSACHE). Guardar como variante propia.
3
Exportar a Excel / CSV
Con la lista en pantalla: Lista → Exportar → Hoja de cálculo local (o usar ícono de exportación). SAP genera un archivo .xlsx directo. También puedes usar la transacción SE16N sobre la tabla AUFK para exportar masivo.
4
Campos de tiempo: calcular Downtime y MTTR
SAP no entrega MTBF/MTTR directo. Necesitas: Tiempo_Parada = F.fin - F.inicio. El tiempo de reparación activo viene del registro de tiempos (IA05/IHPA). Si no tienes ese detalle, usa Duración_OT como proxy de MTTR.
5
Cargar en IndustrialIQ
Arrastra el archivo al recuadro de carga. Si las columnas no coinciden exactamente, el mapeador automático te permite asignar cada columna del archivo al campo correcto. Los cálculos de MTBF, MTTR, disponibilidad y bad actors se recalculan al instante.
6
Otros sistemas compatibles
Maximo: Work Orders → Export (SR/WO type). Oracle EAM: Maintenance Requests → Download. Infor EAM: Equipment Activity → CSV. CMMS manual: Cualquier planilla con las columnas requeridas.
